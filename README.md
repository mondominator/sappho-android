# Sappho

A mobile app for streaming audiobooks from your self-hosted [Sappho server](https://github.com/mondominator/sappho).

## Features

- Stream audiobooks with background playback
- Android Auto support
- Sync progress across devices
- Browse by author, series, and genre
- Dark theme

## Download

- **Android**: [GitHub Releases](https://github.com/mondominator/sappho-android/releases)
- **iOS**: Coming soon

## Quick Start

1. Install the app
2. Enter your server URL (e.g., `https://audiobooks.example.com`)
3. Log in and start listening
